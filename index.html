Objetivo General


En la actualidad el diseño efectivo y eficiente de sitios web puede ser una tarea compleja, difícil y larga. De ahí que las herramientas se conviertan en un elemento clave a la hora de maximizar nuestros recursos.

Sin embargo, hoy por hoy el uso de aplicabilidad de las herramientas es muy amplio y cubre todo tipo de necesidades, por eso resulta difícil escoger la adecuada para cada proyecto.

Nuestro principal objetivo en este proceso es tener un sitio WEB que nos permita conocer y tener información de primera mano sobre el uso de herramientas como GIT y GITHUB para los proyectos.

Adicionalmente el uso de HTML5 y CSS3 para la creación de los contenidos en formato web (Paginas y Sitios Web). En este proyecto nuestra base será un portal que permita conocer de primera mano información descriptiva con lecturas de información básica sobre las herramientas, algunos enlaces multimedia de estos procesos, actividades académicas para refuerzo de los conocimientos previamente adquiridos y datos de los autores de esta página.
